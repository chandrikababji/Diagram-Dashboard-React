# Simple Diagram Dashboard

## Project Description
This project is a React-based Diagram Dashboard that allows users to upload a circuit diagram and view components. The application includes zoom controls and a responsive layout.

## Features

* Upload diagram (PNG/JPG)
* Preview uploaded image
* Zoom In / Zoom Out / Reset controls
* Components list panel
* Responsive layout (desktop and mobile)

## Technologies Used

 React.js
 JavaScript
 CSS
 Flexbox / Grid (for layout)

 Setup Instructions

 1.Git repository link
 https://github.com/chandrikababji/Diagram-Dashboard-React

 2. Install Dependencies

npm install

 4. Start the Development Server

npm start

 5. Open in Browser

http://localhost:3000

Usage

1. Upload a diagram image (PNG or JPG).
2. The diagram will appear in the viewer.
3. Use Zoom In, Zoom Out, and Reset controls.
4. Click components from the list to highlight them.

Responsive Behavior
 Desktop / Tablet → Diagram viewer and component list appear side-by-side.
 Mobile → Layout stacks vertically.
